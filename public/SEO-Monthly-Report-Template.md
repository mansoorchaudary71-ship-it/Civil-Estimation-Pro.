## 🚨 ALERT THRESHOLDS & ESCALATION CHECKLIST

*Trigger immediate audits if any of the following occur:*

**1. Organic Traffic drops >20% Week-over-Week**
*   **Action Checklist:**
    *   [ ] Check Google Search Console for manual penalties or security issues.
    *   [ ] Verify Google Analytics tag is firing correctly (use Tag Assistant).
    *   [ ] Check `robots.txt` for accidental blockages (`Disallow: /`).
    *   [ ] Check server uptime logs for extended downtime.
    *   [ ] Cross-reference with major Google Core Algorithm update dates.

**2. Conversions (PRO Upgrades) drop >30% Week-over-Week**
*   **Action Checklist:**
    *   [ ] Test the Stripe/Payment gateway manually (run a test transaction).
    *   [ ] Check automated email sequences (Are Welcome Emails still sending?).
    *   [ ] Verify the pricing page (`/pro/`) loads within 2 seconds.
    *   [ ] Check if a competitor launched a massive discount / promotion.

**3. Core Keyword Drops out of Top 10**
*   *(e.g., "house construction cost pakistan" falls from Pos #2 to Pos #14)*
*   **Action Checklist:**
    *   [ ] Analyze the exact SERP intent shift — did Google change it to a video carousel or local map pack?
    *   [ ] Audit the competing pages that overtook you (Word count, backlinks, tool UX).
    *   [ ] Refresh content on the target page (update the "2026 Live Rates").
    *   [ ] Fetch and request Indexing in GSC manually.

**4. 404 Errors / Indexing Errors spike >10% in GSC**
*   **Action Checklist:**
    *   [ ] Check recent deployment logs—did a URL structure change?
    *   [ ] Implement 301 redirects immediately for all high-traffic broken paths.
    *   [ ] Resubmit XML sitemap via GSC.
